# Installation Guide

## Prerequisites
- A Linux host system (any distro)
- 10GB+ free disk space
- Internet connection
- Basic Linux knowledge

## Method 1: Stage3 Tarball (Fast)

1. Download stage3
2. Extract to target partition
3. Chroot and configure
4. Install kernel and bootloader

## Method 2: Horizon Bootstrap (From Scratch)

1. Install Astral on host
2. Run Horizon stage 1-3
3. Configure system
4. Reboot

See [Horizon Documentation](horizon) for details.
