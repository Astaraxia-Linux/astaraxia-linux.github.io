Installation Guide

Prerequisites

  • A Linux host system (any distribution).
  • 10GB+ free disk space.
  • Internet connection for fetching source tarballs.

Method 1: Stage3 Tarball (Recommended)

The fastest way to get Astaraxia running.

  1. Download the latest Stage3 tarball.
  2. Extract to your target partition.
  3. Chroot into the environment using astral -c.
  4. Install your kernel and bootloader.

Method 2: Horizon (From Source)

For those who want to build the entire system from scratch (LFS style).

# Run the bootstrap sequence
sudo astral -h --stage 1
sudo astral -h --stage 2
sudo astral -h --stage 3
