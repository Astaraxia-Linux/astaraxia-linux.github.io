Astral Package Manager

The core of Astaraxia: A monolithic, 5,000+ line pure POSIX sh engine designed for absolute control over source builds.

Basic Usage

# Install package
sudo astral -S package-name

# Remove package
sudo astral -R package-name

# Search packages
astral -s keyword

# Update repos
sudo astral -u

Features

  • POSIX Compliance: Runs on any standard shell; no bashisms allowed.
  • Recipe Versatility: Native support for v1, v2, and v3 metadata formats.
  • Dependency Logic: Handles complex trees without the bloat.
  • Git Integration: Build directly from upstream repositories.

For the full technical reference, visit the Astral GitHub Repository.
