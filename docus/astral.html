Astral Package Manager

"Because compiling from source should be less painful than a root canal."

The Engine

Astral is a minimal POSIX package manager for Astaraxia Linux. It builds packages from source because binaries are for people who trust other people's compilers. Think of it as Gentoo's Portage, but written with a focus on POSIX portability and scriptable transparency.

Why Astral?

  • ✓ Source-based: Total silicon optimization.
  • ✓ POSIX-compliant: Runs where Bash doesn't.
  • ✓ Minimal: Just sh, curl, and grit.
  • ✓ 4 Formats: Compatibility from dir to v3.

Why Not?

  • ✗ Time: You actually have a life.
  • ✗ Hardware: You're on a 2008 netbook.
  • ✗ Preference: You like .deb or .rpm.
  • ✗ Normality: You aren't a terminal maniac.

Installation

To inject Astral into your host system:

curl -O https://raw.githubusercontent.com/Astaraxia-Linux/Astral/main/astral
chmod +x astral
sudo mv astral /usr/bin/

Basic Usage

Sync and build a package:

sudo astral -S [package]

Remove and clean orphaned files:

sudo astral -R [package]
