Bootstrap from Scratch

Total control. Build your entire system from source using the Horizon engine.

# Install Astral
curl -O https://raw.githubusercontent.com/.../astral
chmod +x astral
sudo mv astral /usr/bin/

# Initialize Horizon
export LFS=/mnt/lfs
sudo astral -h --init

# Run stages (12-20 hours)
sudo astral -h --stage 1
sudo astral -h --stage 2
sudo astral --chroot $LFS
astral -h --stage 3
