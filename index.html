What is Astaraxia?

A POSIX-compliant, source-based Linux distribution built with simplicity and control in mind.

Features

  • Astral package manager (pure POSIX sh)
  • LFS-style bootstrap with Horizon
  • Three recipe formats (v1, v2, v3)
  • No systemd dependency

Quick Start

curl -O https://raw.githubusercontent.com/Astaraxia-Linux/Astral/main/astral
chmod +x astral
sudo mv astral /usr/bin/
sudo astral -h --init
