What is Astaraxia?

Astaraxia is a POSIX-compliant Linux distribution designed for users who want to understand their system from the ground up. No bloat, no unnecessary abstractionsâ€”just you and the source.

Features

  • Astral Manager: A 5,000+ line pure POSIX shell package manager.
  • Horizon Bootstrap: Automated 3-stage LFS-style deployment.
  • Recipe Versatility: Native support for v1, v2, and v3 metadata formats.
  • Init Freedom: No systemd dependency; built for modularity.

Quick Start

Get Astral running on your current system to start building:

# Download and install Astral
curl -O https://raw.githubusercontent.com/Astaraxia-Linux/Astral/main/astral
chmod +x astral
sudo mv astral /usr/bin/

# Initialize the Horizon bootstrap environment
export LFS=/mnt/lfs
sudo astral -h --init
